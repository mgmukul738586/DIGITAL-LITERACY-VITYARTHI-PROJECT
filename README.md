## Project Overview
The primary objective of this project is to assist peers in navigating the digital world, focusing on building professional online identities and ensuring safe, responsible communication. The portfolio documents the completion of five distinct tasks designed to develop foundational digital tools and behaviors.

### Task 1: Digital Literacy Awareness Infographic
Created a visual resource using **Canva** to help students understand the core pillars of digital literacy.
* **Topics Covered:** Safe internet practices, source verification, and professional email etiquette.
* **Key Insight:** Balanced icons and text to condense complex information into a single, digestible page.

### Task 2: Student Digital Portfolio
Established a professional digital footprint across multiple platforms:
* **GitHub:** Created a profile README outlining academic goals and technical skills.
* **LinkedIn:** Updated education details to serve as a professional networking hub.
* **HackerRank:** Set up a profile to showcase coding progress and technical proficiency.

### Task 3: Coding & Collaboration Platforms
Engaged with technical and collaborative tools to build academic readiness:
* **Coding:** Completed a beginner-level Python challenge on HackerRank.
* **Collaboration:** Developed a "Digital Literacy Awareness Quiz" using **Google Forms** and linked it to **Google Sheets** for data management practice.

### Task 4: Professional Email & Etiquette
Focused on formal communication and social media responsibility:
* **Drafted Emails:** Created templates for requesting assignment extensions and inquiring about summer internships.
* **Social Media Checklist:** Compiled 10 "Do's and Don'ts," emphasizing the **"Billboard Test"** and the permanence of digital footprints.

### Task 5: Cybercrime Awareness & Prevention
Researched **UPI/Online Payment Fraud**, specifically the "Pending Utility Bill" scam.
* **Findings:** Documented how social engineering is used to trick victims into revealing UPI PINs.
* **Prevention:** Developed a "Stay Safe Online" guide with eight actionable tips, such as using mobile data instead of public Wi-Fi for financial transactions.
